كلاود منيو v2 — دليل النشر بالسحب على Netlify
=============================================

هذا المجلد (deploy/) هو الموقع الجاهز كاملاً. لنشره:

1) افتح https://app.netlify.com → Sites
2) اسحب مجلد deploy/ كاملاً وأفلته في صفحة الموقع (Deploys → drag & drop)
3) انتهى! ملفات _redirects و _headers تُطبَّق تلقائياً.

ملاحظات:
- لا تسحب ملفات فردية — اسحب المجلد كاملاً حتى تعمل روابط المطاعم /<slug>.
- الموقع متصل بقاعدة Supabase الحالية: حساباتك ومطاعمك تعمل مباشرة.
- الدفع حالياً بمفتاح Moyasar تجريبي (pk_test) — قبل الإطلاق الفعلي استبدله
  في app/src/lib/config.ts ثم أعد البناء (npm run build داخل مجلد app/).

لأي تعديل مستقبلي: الكود المصدري في مجلد app/ — عدّل ثم npm run build
وسيتحدّث محتوى deploy/ تلقائياً، ثم اسحب deploy/ من جديد.
